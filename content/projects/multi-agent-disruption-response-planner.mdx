This project won first place because it treated AI as orchestration, not just output generation.

## System shape

- Agents handled different parts of the planning workflow
- Backend services preserved state and constraints across steps
- The frontend made scenario comparisons interactive instead of opaque

## Engineering takeaway

The best part of the project was making LLM-driven reasoning auditable enough to support real decision discussion.
