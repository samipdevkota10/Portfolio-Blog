This project was a good forcing function for backend fundamentals: concurrency, fanout, persistence, and reliability under load.

## Core work

- Architected the WebSocket communication layer
- Added Redis-backed message distribution for throughput
- Modeled chat persistence and presence in PostgreSQL

## Why I like it

Realtime systems make tradeoffs visible. You feel every weak assumption in delivery guarantees, fanout strategy, and connection lifecycle handling.
